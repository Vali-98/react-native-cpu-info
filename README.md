# @vali98/react-native-cpu-info

A module for getting cpu info

## Installation

```sh
npm install @vali98/react-native-cpu-info
```

## Usage


```js
import { getThreads } from '@vali98/react-native-cpu-info';

// ...

const result = getThreads(); // number
```
